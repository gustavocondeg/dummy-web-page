# API

## Environment specification

node 16.x
npm 8.x

## API prepare

- cd mes-api
- npm i

## API start development

- cd mes-api
- npm run start:dev

this start API on port 3000 - http://localhost:3000/api

## API run test

- cd mes-api
- npm run test

The report is generate in folder /mes-api/reports

## API run coverage

- cd mes-api
- npm run test:cov

The report is generate in folder /mes-api/reports/coverage

## API generate documentation

- cd mes-api
- npm run doc

The documentation is generate in /mes-api/documentation

## API build for production

- cd mes-api
- npm run build

The API build is generate in /mes-api/dist

# React Aplication

## App prepare

- cd demo-app
- cd npm i

## App start development

- cd demo-app

## App run test

- cd demo-app

The report is generate in folder /demo-app/reports

## App build for production

- npm run build

The app build is generate in /demo-app/dist
